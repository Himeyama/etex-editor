LaTeX のエディター

## インストール方法

### 依存
WSL2 + texlive のインストールが必要。

### 対応

- Windows 10
- Ubuntu (ビルドは未)

### ビルドとインストール
```sh
npm run build-win
```

`dist/chocho-editor x.x.x.exe` をクリックしインストール
